### Secrets requerits
| Nom            | Què conté                |
|----------------|--------------------------|
| `GITHUB_TOKEN` | PAT amb scope **repo**  |

### Setup
Codex clona el repo (HTTPS) ➜ executa `setup.sh` ➜ el remote `origin` ja queda
apuntant a HTTPS amb el token. No cal fer `git fetch/pull`.
